# 🌾 Agrimitra

**Agrimitra** is an intelligent agriculture assistant designed to help farmers with crop management, disease detection, and smart decision-making using AI and real-time insights.

---

## 🚀 Features

- 🌱 AI-based Crop Guidance  
- 🦠 Plant Disease Detection (Image + Text)  
- 📅 Crop Calendar & Planning  
- 📊 Market Price Insights  
- 🌦️ Weather Integration  
- 💬 Smart AI Assistant (Plant Doctor)  
- 🌐 Multilingual Support (Hindi, Marathi, English)  
- 📷 Image-based Diagnosis  

---

## 🛠️ Tech Stack

- **Flutter** (Frontend)  
- **Dart**  
- Gemini API (AI Assistant)  
- Plant.id API (Plant identification & disease detection)  
- Firebase (Auth / Backend)  
- OpenWeather API
